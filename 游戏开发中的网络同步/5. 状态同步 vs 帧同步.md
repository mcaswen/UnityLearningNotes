#### 一、核心差异总结

| 特性        | 帧同步                 | 状态同步               |
| --------- | ------------------- | ------------------ |
| **同步内容**  | **操作指令**            | **游戏状态**           |
| **服务端角色** | **消息路由器**           | **权威模拟器**          |
| **客户端角色** | **完整模拟器**           | **表现的渲染器**         |
| **确定性要求** | 极高，必须绝对一致           | 较为宽松，服务端权威保证       |
| **网络流量**  | 低且稳定，与玩家操作相关        | 高且波动，与游戏复杂度相关      |
| **安全性**   | 较低，逻辑在客户端           | 极高，逻辑在服务端          |
| **流畅性**   | 操作反馈块，但可能整体卡顿       | 有一定延迟，但整体流畅性高      |
| **适用游戏**  | RTS、MOBA、棋牌、需要回放的游戏 | MMORPG、FPS、逻辑复杂的游戏 |
